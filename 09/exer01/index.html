Criar sua primeira página da WEB HTML

Ilustra as técnicas básicas de programação usando o Visual Studio Code para a escrita de HTML.

Objetivos de aprendizagem

Depois de concluir este módulo, você poderá:

  • Descrever os conceitos básicos de HTML
  • Usar uma ferramenta de codificação
  • Criar uma página HTML básica
  • Criar listas vinculadas em HTML

Pré-requisitos

Conhecimento básico da terminologia computacional.
